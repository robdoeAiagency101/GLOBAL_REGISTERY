# Earnings Model

This repository uses GitEarn for contributor rewards.

## How You Earn
1. Connect your GitHub + Wallet to GitEarn
2. Get assigned to an issue
3. Submit a pull request referencing the issue
4. After merge, GitEarn calculates your reward
5. Payment is sent to your connected wallet

## Weighting
- Code contributions: 60%
- Issue resolution: 20%
- Documentation: 10%
- Testing: 10%

## Identity
Your NFT domain + wallet serve as your contributor identity.
