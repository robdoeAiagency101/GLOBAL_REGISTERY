# GitEarn Licence Extension

This project uses the Apache2.0 licence with an additional GitEarn revenueshare layer.

## Earnings
Contributors earn based on:
- Pull requests merged
- Issues resolved
- Code quality and impact
- Skill execution inside the AiFACTORi engine
- Oracle or mesh contributions

## Identity
All earnings are tied to the contributors:
- Wallet address
- NFT domain identity
- GitEarn account

## Payouts
Payouts are sent directly to the wallet linked to the contributors GitEarn profile.

## Bounties
Issues may include GitEarn bounties using the /bounty  command.
The GitEarn bot manages confirmation, assignment, and payout.
